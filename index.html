O que é HTML?


HTML é a sigla para Hyper Text Markup Language ou, linguagem de marcação hiper texto.

HTML não é uma linguagem de programação, ela serve para estruturar a página na web, utilizando elementos que dizem ao navegador como exibir o conteúdo da página.

  2. Parágrafo


    voltar

    Utilizamos a tag < p > para indicar o início e o final de um parágrafo. Logo, todo o texto que está entre estas tags é considerado um parágrafo.


  3. TAGs para formatação de textos


    voltar
    • Tag < mark > é utilizada para marcação.
    • Tag < small > é utilizada para diminuir um texto.
    • Tag < i > é utilizada para deixar um texto itálico.
    • Tag < u > deixa um texto sublinhado.
    • Tag < sup > coloca o textopara cima.
    • Tag < sub > coloca o textopara baixo.
    • Tag < strong > define um texto como importante, geralmente deixando em negrito.
    • Tag < b > similar ao strong, deixa o texto em negrito.
    • Utilizamos a Tag < blockquote > para uma citação. Dessa forma:
      Esta é uma citação.
    • Utilizamos também a Tag < abbr > para colocar uma abreviação, por exemplo: HTML é legal!
    • Podemos utilizar a tag < del > para exibir um texto riscado.
